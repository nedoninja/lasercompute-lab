# lasercompute-lab

## How to start using 
``` bash
    git clone https://github.com/nedoninja/lasercompute-lab
    cd lasercompute-lab/
    python -m venv venv         # if use *NIX = python3      
    source venv/bin/activate    # if use *NIX
    venv\Scripts\activate.bat   # if use windows
    pip install -r requirements.txt
    python main.py
```
